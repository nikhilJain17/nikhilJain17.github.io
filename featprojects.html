Side Projects

Here are some projects I've worked on at hackathons and competitions. For more, check out my Github or my Devpost profiles.

  • Tacteyele
    PennApps XV 3rd Place
    Tacteyele is a hands-free way to use your computer. Users tilt their head to move the mouse, wink to click, and speak to type or do keyboard shortcuts.

    Tacteyele runs on Python 2.7, OpenCV, and Watson SPT.

    View the code on Github.
    Watch the demo.
    Read more on Devpost.
  • PebbleTennis
    HackRU Fall 2015 2nd Place, Best Hardware Hack, Best Use of SendGrid, Best Use of Pebble
    PebbleTennis is a smart tennis racket that uses accelerometer data to track your strokes. It sends the data to a server, which emails the data to you and your coach. At HackRU, my friend and I (who both play tennis) attached a Pebble smartwatch onto a tennis racket and used the Pebble's accelerometer.

    PebbleTennis runs on Node.js and Pebble.js.

    View the code on Github.
    Watch the demo.
    Read more on Devpost.
  • Door Sight
    Engineering Capstone Project
    Door Sight is a facial recognition door lock. It uses a Raspberry Pi and camera to control the door and talk to a server where users control admin settings. This is a work in progress, so stay tuned for updates.

    View the code on Github.
  • Drivr
    ProfHacks 3rd Place, Best Quantified Self
    Drivr is a system to make driving safe. Drivr can detect if you are swerving (from accelerometer data in the Pebble smartwatch), if you let go of the wheel (from the Myo armband), and if you are paying attention to the road (from the Muse headband). Drivr displays all this information, including speed and current music playing, in a compact heads-up display. It vibrates if you aren't paying attention, and lets you change the music by flicking your wrist.

    Drivr runs Swift and Node.js on the Myo armband, Muse headband, and Pebble smartwatch.

    View code on Github.
    Read more on Devpost.
  • Robn
    Congressional App Challenge 2016
    Robn turns your phone into a wireless mouse and keyboard for your laptop.

    Robn runs on Android and iOS. It also uses Node.js and Python.

    View code on Github.
    Watch the demo.
  • Spare Change
    HackCooper 2016
    Spare Change uses image recognition to detect how much spare change you have. Hold your coins in your hand, put your phone over it, and it will tally up the total alongside your previous balance.

    Robn runs on Android, Java, OpenCV, and Node.js.

    View code on Github.
    Read more on Devpost.
