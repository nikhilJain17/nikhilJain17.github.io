Battleship

Battleship was a project for my electrical engineering class. My group and I made a 2 person replica of the battleship game using Java Swing for the GUI, and Socket.io plus node.js for the backend server. We initially wanted to use qt with C++ but found Swing much easier to configure and work with. Any two players can play the game wirelessly since we used a server to handle the game logic. This project taught me a lot about server side programming and HTTP requests. You can check out the code here.
