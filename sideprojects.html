Side Projects

Here are some smaller side projects I've worked on for fun on my own. For more, check out my Github or my Devpost profiles.

  • midPoint
    midPoint is an app that recommends the best place to meet with your friends based on location and type of place. It makes planning things easier.

    midPoint started as a hackathon project in late 2014, when I barely knew any programming. My team and I continued working on it over the course of several months, and released it on the App Store and Google Play Store.

    midPoint runs on Android, iOS, and Node.js.
    View the code for Android and the code for iOS.
    Download it for iOS or Android.
  • Haskell Email Bot
    Haskell Email Bot is a command-line program that lets you send emails from anybody to anybody. You can choose the subject, title, from-address, and to-address.

    Email Bot runs on Haskell and Sendgrid.
    View the code on Github.
  • BD Calculator
    A program for a 1 digit by 1 digit calculator written in MPASM for the PIC16F887 microcontroller. It supports addition, subtraction, multiplication, and division. Additionally, it catches dividing by 0 and supports negative answers for subtraction.

    BD Calculator runs Assembly on a PIC16F887 microcontroller, which interfaces to a circuit with LED outputs and button inputs.

    View the code on Github.
  • What's Up Dawg?
    What's Up Dawg is a Facebook Messenger bot for paired rapping. Message it a line, and it will send one back that rhymes any time. I made this when Messenger bots just came out to have some fun.

    What's Up Dawg runs on Node.js and is hosted on Heroku.

    View the code on Github.
    Try it out here. Note that it might be slow or buggy due to hosting.
  • Silver Browser
    Silver Browser is a basic web browser. It delivers HTML and some crude CSS to your fingertips.

    Silver Browser runs on Java and Java Swing.

    View the code on Github.
  • Siri Clone for Android
    This Android app is a voice assistant that can perform some basic tasks on your phone. It's like Siri, but for Android.

    Siri Clone for Android runs on Java and Android 5+.

    View the code on Github.
